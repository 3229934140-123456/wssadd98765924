## 1. 产品概述

冷藏车厢温度追踪 Web 看板，为冷链调度中心值班员提供多车并行运输的实时监控与异常处置能力。解决"谁异常、异常多久、谁处理过"三个核心问题，保障冷链货物运输安全。

- 目标用户：冷链调度中心值班员
- 核心价值：多车并行监控、异常快速定位、处置全程留痕

## 2. 核心功能

### 2.1 用户角色

| 角色 | 登录方式 | 核心权限 |
|------|----------|----------|
| 调度值班员 | 账号登录 | 车辆监控、轨迹回放、异常处置、交接记录查看 |

### 2.2 功能模块

1. **车辆列表页**：筛选条件、在途车辆列表、实时状态概览
2. **单车轨迹页**：地图轨迹、温度时间轴、阶段回放、超温标注
3. **异常处置页**：异常列表、详情面板、处理动作、交接记录

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 车辆列表页 | 顶部筛选栏 | 按线路、承运商、货品类型、温度状态筛选在途车辆 |
| 车辆列表页 | 统计概览卡 | 在途总数、正常数、预警数、异常数实时统计 |
| 车辆列表页 | 车辆列表 | 车牌号、当前温度、设定温区、最近上报时间、门磁状态、承运商、线路 |
| 车辆列表页 | 状态标签 | 正常（绿色）、预警（橙色）、异常（红色）三色状态标识 |
| 单车轨迹页 | 地图区域 | 展示车辆行驶轨迹，位置点标注温度值 |
| 单车轨迹页 | 时间轴滑块 | 拖动滑块回放运输全过程，定位装货/等待/卸货阶段 |
| 单车轨迹页 | 温度曲线 | 时间-温度折线图，温区上下限参考线，超温段高亮 |
| 单车轨迹页 | 车辆信息卡 | 车牌号、司机、电话、承运商、货品、温区设定 |
| 异常处置页 | 异常列表 | 按时间倒序展示温度越界事件，含车辆、时长、状态 |
| 异常处置页 | 异常详情 | 异常开始/结束时间、持续时长、影响车厢、最高/最低温度 |
| 异常处置页 | 处理动作 | 已通知司机、已联系客户、转入复核、已恢复正常等选项 |
| 异常处置页 | 交接记录 | 处理人、处理时间、处理结果、备注信息的完整记录链 |

## 3. 核心流程

值班员登录系统 → 查看车辆列表概览 → 筛选关注车辆 → 发现异常车辆 → 进入单车轨迹回放确认 → 跳转异常处置 → 选择处理动作并填写备注 → 生成交接记录 → 异常闭环

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深蓝灰底色（#0F172A），工业监控风格，适合长时间盯屏
- **强调色**：正常绿（#10B981）、预警橙（#F59E0B）、异常红（#EF4444）、信息蓝（#3B82F6）
- **按钮风格**：扁平化、直角微圆角、状态色填充、悬停微提亮
- **字体**：JetBrains Mono 等宽字体用于数据展示，Inter 用于界面文字；数字使用等宽字体便于快速比对
- **布局风格**：卡片式分区、高密度信息展示、左侧导航 + 右侧内容区
- **图标风格**：线性图标（Lucide），简洁专业

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 车辆列表页 | 顶部筛选栏 | 下拉选择器、搜索框、状态标签筛选、高密度紧凑布局 |
| 车辆列表页 | 统计卡片 | 大字号数据、状态色标识、微动效数据更新 |
| 车辆列表页 | 车辆列表 | 表格布局、行悬停高亮、状态色条、门磁开关图标 |
| 单车轨迹页 | 地图区域 | 深色底图、轨迹线、温度色标点、当前位置脉冲动画 |
| 单车轨迹页 | 时间轴 | 可拖动滑块、阶段分段标记、超温段红色高亮 |
| 单车轨迹页 | 温度曲线 | 面积图、温区参考线、超温区域红色填充 |
| 异常处置页 | 异常列表 | 时间线布局、异常卡片、处理状态标签 |
| 异常处置页 | 详情面板 | 键值对数据展示、司机电话一键拨打、处理按钮组 |

### 4.3 响应式

- 桌面端优先设计，适配 1920×1080 及以上分辨率
- 左侧导航固定宽度，内容区自适应
- 表格区域支持横向滚动
- 最低支持 1280px 宽度

### 4.4 动效与交互

- 数据刷新时数字滚动过渡
- 异常状态脉冲呼吸动画
- 页面切换淡入淡出
- 表格行悬停背景微变
- 时间滑块拖动实时反馈
